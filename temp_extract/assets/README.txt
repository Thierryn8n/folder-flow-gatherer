FISCAL FLOW PRINTER - INSTRUÇÕES PARA ÍCONE

Para adicionar um ícone personalizado ao aplicativo, siga estas etapas:

1. Crie ou obtenha um arquivo de ícone (.ico) para o aplicativo
2. Renomeie o arquivo para "icon.ico"
3. Coloque o arquivo nesta pasta (assets)

Um ícone no formato .ico é necessário para:
- Ícone do aplicativo na barra de tarefas
- Ícone na bandeja do sistema (tray)
- Ícone do instalador

Recomendações:
- Resolução mínima: 256x256 pixels
- Formato: .ico (pode conter múltiplos tamanhos)
- Cores: Fundo transparente para melhor visualização

Se você não adicionar um ícone personalizado, será utilizado um ícone padrão.
